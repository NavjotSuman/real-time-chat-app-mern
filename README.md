### Setup .env file

```js
PORT =...
MONGODB_URI =...
JWT_SECRET =...

SERVER_ORIGIN =... // Backend Origin

# cloudinary api keys
CLOUDINARY_CLOUD_NAME =...

```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
